# Smart Campus Crowd Intelligence System

An intelligent, real-time campus crowd monitoring and analysis system designed to optimize campus facility usage and improve student safety using Computer Vision and Data Analytics.

## Overview
The Smart Campus Crowd Intelligence System leverages live camera feeds and historical data to provide real-time occupancy insights. It helps administrators and students identify peak hours, manage crowd density, and optimize resources like libraries, canteens, and administrative offices.

##  Key Features
- **Real-time Monitoring**: Live crowd counting using computer vision.
- **Trend Analysis**: Visual representations of crowd density over hours, days, and weeks.
- **Smart Recommendations**: Suggests the best time to visit a location based on predicted low traffic.
- **Multi-Location Support**: Monitor multiple areas like the Library, Canteen, and Labs from a single dashboard.
- **Intelligent Insights**: Dynamic feedback on peak hours and crowd trends.
- **Dual Data Source**: Seamlessly toggle between live camera feeds and simulated data for testing.

##  Tech Stack
- **Frontend**: React.js, Recharts, Lucide React, Vanilla CSS.
- **Vision**: Python, OpenCV (Processing layer).
- **Backend**: (In-development) Node.js / Python FastAPI.

## Project Structure
- `frontend/`: React-based dashboard for visualization.
- `backend/`: API services and data management.
- `vision/`: Computer vision scripts for crowd detection.
